# 3D Printed NAS

Watch the build video ↓

[<img src="/Images/3D-Printed-NAS_Thumbnail.jpg">](https://youtu.be/NXIu-B52WPU)

I was juggling too many small SSDs for backups, so I set out to build something more permanent.
